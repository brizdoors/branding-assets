Briz Doors SVG Usage Guide

Use SVG as the primary web asset format. These files have transparent backgrounds by default.

Back to asset browser

Direct image tag (recommended default)

<img
  src="https://brizdoors.github.io/branding-assets/branding/Briz%20Doors%20Branding/icon-dark-512.svg"
  alt="Briz Doors icon"
  width="128"
  height="128"
/>

Inline SVG (best for styling parts of the mark)

<!-- Paste raw SVG markup inline in your HTML -->
<svg viewBox="0 0 512 512" aria-label="Briz Doors icon" role="img">
  ...
</svg>

CSS background image

.brand-mark {
  width: 160px;
  height: 36px;
  background: url("https://brizdoors.github.io/branding-assets/branding/Briz%20Doors%20Branding%204/primary-dark-512.svg")
    no-repeat center / contain;
}

React / Next.js usage

export function BrandIcon() {
  return (
    <img
      src="https://brizdoors.github.io/branding-assets/branding/Briz%20Doors%20Branding/icon-dark-96.svg"
      alt="Briz Doors"
      width={96}
      height={96}
    />
  );
}
Keep icons/logos on transparent background. For raster fallbacks, export PNG versions with transparency preserved.
